# Species Observation Trends from Camera Trap Data

## Overview
This project analyzes species-wise observation trends from camera trap data to assess temporal accumulation patterns and sampling sufficiency. The analysis builds on cleaned camera trap datasets and demonstrates ecological interpretation using cumulative observation curves.

## Objectives
- Visualize species observation trends
- Assess sampling sufficiency using cumulative counts
- Identify dominant vs low-detection species

## Data
- Source: Open-access camera trap dataset Mendeley Data repository (processed in [camera-trap-data-analysis](https://github.com/Ankanhalder2000/camera-trap-data-analysis))
The data is used strictly for academic and skill demonstration purposes 
- Input file: `camera_trap_cleaned.csv`

## Methods
- Data wrangling using pandas
- Species-wise aggregation
- Cumulative count computation
- Line plots for trend visualization

## Key Findings
- Dominant species (Impala, Zebra) show sustained accumulation
- Rare species reach early stabilization
- Results indicate incomplete sampling for rarer taxa

## Limitations
- No temporal metadata available
- Spatial clustering limits habitat inference
